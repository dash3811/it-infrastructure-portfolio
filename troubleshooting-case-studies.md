# Troubleshooting Case Studies

Use this section to demonstrate technical problem solving. Each case study should be short, specific, and sanitized.

## Case Study Template

```text
## Case Study: [Problem Title]

### Problem
What was broken?

### Symptoms
What did you observe?

### Diagnosis
How did you investigate?

### Tools Used
What tools, commands, dashboards, or logs did you use?

### Root Cause
What caused the problem?

### Resolution
How did you fix it?

### Prevention
What changed to prevent recurrence?
```

## Suggested Case Studies

- DNS resolution issue
- Reverse proxy / SSL issue
- Docker container networking issue
- Home Assistant automation issue
- VLAN client assignment issue
- Synology degraded storage warning
- Roomba/Home Assistant device integration issue
- Camera/Frigate performance issue
