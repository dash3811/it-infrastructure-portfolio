# IT Infrastructure Portfolio

This portfolio documents a self-hosted infrastructure lab built to develop and demonstrate hands-on skills in systems administration, networking, virtualization, Linux, Docker, storage, monitoring, automation, and technical troubleshooting.

The environment is intentionally documented using sanitized examples. No private IP addresses, public IPs, credentials, tokens, serial numbers, device IDs, home addresses, or sensitive screenshots are included.

## Environment Summary

- **Virtualization:** Proxmox VE
- **Storage:** Synology NAS
- **Operating Systems:** Ubuntu, Debian
- **Containerization:** Docker / Docker Compose
- **Networking:** Ubiquiti UniFi, VLANs, DNS, DHCP, VPN, firewall rules, port forwarding
- **Remote Access & Proxying:** Cloudflare, Nginx Proxy Manager, SSL certificates, reverse proxies
- **Automation:** Home Assistant, YAML automations, smart-home integrations
- **Monitoring:** Grafana, VictoriaMetrics
- **Security / DNS Filtering:** Pi-hole, Unbound
- **Scale:** 20+ Docker containers, 2 VMs, 3 VLANs, 44 connected smart devices
