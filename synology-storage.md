# Synology Storage

## Overview

Synology NAS is used for centralized storage, application data, backups, media storage, and infrastructure support.

## Skills Demonstrated

- Managing NAS storage
- Organizing shared folders
- Supporting Docker application data
- Planning backups and recovery
- Troubleshooting storage availability
- Managing permissions and access

## Storage Responsibilities

- Maintain storage for application configuration and persistent data
- Organize media and service directories
- Support backup workflows
- Monitor disk health and storage usage
- Troubleshoot degraded volume or drive warnings
