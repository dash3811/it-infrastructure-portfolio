# GitHub Setup Guide

## 1. Create a GitHub account or sign in

Go to GitHub and create/sign in to your account.

## 2. Create a new repository

Recommended repository name:

```text
it-infrastructure-portfolio
```

Description:

```text
Sanitized IT infrastructure portfolio documenting virtualization, networking, Docker, Linux, storage, monitoring, and automation experience.
```

Recommended settings:

- Public repository
- Add README: No, because this starter package already includes one
- Add .gitignore: Optional
- License: Optional

## 3. Upload files using the browser

Beginner-friendly method:

1. Open the new repository
2. Click **Add file**
3. Click **Upload files**
4. Drag the contents of this folder into GitHub
5. Click **Commit changes**

## 4. Review before publishing

Before uploading, search every file for private information:

```text
password
token
secret
api
key
public ip
domain
address
phone
email
serial
mac
```

## 5. Add the portfolio to your resume later

Once the repo is cleaned up and published, add it to your resume header:

```text
GitHub: github.com/yourusername/it-infrastructure-portfolio
```
