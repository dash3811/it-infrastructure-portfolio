# Proxmox Virtualization

## Overview

Proxmox VE is used to host virtual machines for lab services, Linux administration practice, and infrastructure workloads.

## Environment

- 2 virtual machines
- Ubuntu and Debian Linux environments
- Used for testing, service hosting, and infrastructure administration practice

## Skills Demonstrated

- Creating and managing virtual machines
- Allocating CPU, memory, and storage resources
- Installing Linux operating systems
- Managing VM lifecycle: start, stop, reboot, snapshot, restore
- Troubleshooting VM networking and service availability
- Planning host resources and storage usage
